### Установка и запуск проекта
— установка стабильной версии:
npm install

— сборка проекта:
npm run build

— запуск версии для разработчика:
npm run dev

— запуск версии на Express (приложение запуститься по адресу http://localhost:3000):
npm run start

— предпросмотр проекта:
npm run preview
